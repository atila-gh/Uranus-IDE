**Uranus IDE** is a lightweight, extensible Python development environment inspired by Jupyter and built with PyQt5. It offers a modular, cell-based interface for writing, executing, and documenting Python code â€” with full support for multilingual workflows and right-to-left (RTL) languages.

Uranus IDE is designed to empower developers, educators, and students across the world. 
It supports Unicode natively and handles RTL scripts with precision, making it ideal for communities working in:

- **Arabic**

- **Divehi (Maldivian)**  
- **Yiddish**

> âš ï¸ **Why Uranus?**  
> The creation of Uranus IDE was directly motivated by the lack of proper RTL support in mainstream tools like Jupyter.
 Despite its popularity, Jupyter has long overlooked the needs of developers working in right-to-left languages.
 Uranus aims to fill this gap by offering a truly RTL-aware Python IDE.  
>  
> We hope this project becomes a collaborative platform for developers from RTL language communities to contribute, extend, and
 benefit from a tool that finally speaks their language â€” both literally and structurally.

## ðŸ”§ Features That Make Uranus Unique

- Cell-based architecture with separate code and document editors  
- IPython kernel integration with threaded execution  
- Rich output rendering: text, images, tables  
- Python syntax highlighting with multi-line string and scientific structure support  
- Export to Jupyter-compatible `nbformat`  
- Custom license and secure publishing workflows  
- Native support for RTL languages and multilingual documentation
